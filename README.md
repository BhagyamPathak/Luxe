# 🛍️ LUXE — Premium E-Commerce Store

A modern, fully responsive **e-commerce landing page UI** built using **HTML, CSS, and JavaScript**.  
LUXE is designed with a luxury aesthetic, smooth animations, and a clean shopping experience layout.

---

## ✨ Features

- 🎨 Premium modern UI with dark luxury theme
- 📱 Fully responsive design (mobile + tablet + desktop)
- 🛒 Product grid with pricing, discounts & badges
- ⏳ Live countdown flash sale timer
- ❤️ Wishlist & Add-to-Cart interactive buttons
- 📦 Feature highlights section (shipping, support, savings, etc.)
- 📢 Promotional banners & seasonal deals section
- 📬 Newsletter subscription UI
- 🔻 Fully structured footer with social links & payment icons
- ⚡ Smooth hover animations & micro-interactions

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Custom Variables + Flex/Grid)
- JavaScript (Vanilla)
