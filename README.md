# 🛍️ LUXE — Premium E-Commerce UI

A modern, fully responsive **frontend e-commerce website** built using pure **HTML, CSS, and JavaScript**.  
LUXE is a premium UI concept for an online store with smooth animations, elegant design, and interactive components.
